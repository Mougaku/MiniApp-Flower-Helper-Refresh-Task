# MiniApp-Flower-Helper-Refresh-Task (花札物语刷新招待助手) 🚀

这是一个基于 [AutoX.js](https://github.com/automan-bot/AutoX) 开发的安卓自动化脚本。它利用 GMLKit OCR (光学字符识别) 技术，能够自动在小程序或 App 的多个任务栏（Tab）之间切换，直到所有任务栏都刷新出你指定的目标任务。

## 🌟 功能特性

- **多目标匹配**：支持同时设置多个目标任务名（如“白银生晖”、“蓝絮映雪”）。
- **流转刷新逻辑**：通过切换任务栏触发刷新的机制，无需繁琐的重复返回操作。
- **OCR 驱动**：不依赖于控件 ID，只要屏幕上有字就能识别，兼容性极强。
- **动态适配**：支持任意数量的任务栏（Tab），仅需配置坐标即可。
- **安全模拟**：内置随机延迟与震动反馈，模拟真人操作。

## 📸 工作流程

1. **扫描**：自动点击 1 号任务栏。
2. **识别**：通过 GMLKit OCR 获取当前任务名称。
3. **判断**：
   - 如果是目标任务：计数并切换到下一个任务栏。
   - 如果不是目标任务：点击“拒绝”，切换到下一个任务栏触发刷新。
4. **完成**：当所有任务栏连续达标时，脚本自动停止并震动提醒。

## 🛠️ 安装与使用

### 环境准备
1. 下载并安装 [AutoX.js](https://github.com/automan-bot/AutoX/releases)。
2. 在 AutoX.js 设置中开启**无障碍服务**、**悬浮窗权限**和**前台服务**。
3. 确保安装了 **OCR 扩展插件**（首次运行脚本时通常会提示下载）。

### 配置脚本
1. 将本项目中的 `main.js` 代码复制到 AutoX.js 中。
2. 在代码开头的 `CONFIG` 部分修改以下参数：
   - `TARGET_TASKS`: 你想要的任务名称列表。
   - `TABS`: 任务栏切换按钮的坐标。
   - `REJECT_BTN`: “拒绝”或“放弃”按钮的坐标。

### 运行
点击 AutoX.js 的运行按钮，切换到目标小程序页面即可开始。

##  备注
坐标确认：由于不同设备坐标可能不一致，请确保你的坐标 [550, 2300] 等依然准确。如果屏幕比例不一样，记得用“开发者选项-指针位置”重新量一下。
文字匹配：代码里的 白银生晖 必须和屏幕上显示的文字一模一样。如果 OCR 经常把某个字认错（比如把“晖”认成“辉”），你可以把 TASK 改成 白银生（只匹配前三个字），这样容错率更高。
